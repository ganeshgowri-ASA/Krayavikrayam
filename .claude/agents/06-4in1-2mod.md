# Agent 06 — 4-in-1 Combo Chamber (TC + HF + DH + UV, 2 module)

## Operating modes

- **MODE A** — Thermal Cycling −40 °C to +85 °C (TC, MQT 11).
- **MODE B** — Humidity Freeze (TC + 85 % RH, MQT 12).
- **MODE C** — Damp Heat 85 °C / 85 % RH (MQT 13).
- **MODE D** — UV preconditioning 60 °C + UV array (MQT 10).

## Geometry

- Outer: 1800 L × 1200 W × 2000 H mm.
- Inner: 1400 L × 800 W × 1600 H mm (2 modules vertical).
- UV lamp array (retractable or side-mounted; covers module area).
- Combined refrigeration + heating + humidification unit (right-side bay).
- UV door window: UV-transparent quartz glass 300 × 200 mm (MODE D only).
- UV safety interlock disables UV when door open.
- Controller: multi-mode PLC with mode selector.

## HVACR design (cascade refrigeration)

Per the attached chat 4-in-1 master plan, use cascade refrigeration to reach
−45 °C evaporator with 40 °C condenser:
- High stage: R507A (or R134a if commercial availability easier in IN).
- Low stage: R23 (or R508B for higher COP).
- Heat exchanger between stages: brazed plate.
- Compressor selection: BITZER (cascade application supported in BITZER
  Select software).

Use DWSim + CoolProp to size:
- Compressor pressure ratio per stage
- Mass flow per stage
- COP overall
- P-H diagram per stage
- Component selection table (compressors, condenser, evaporator, EXVs, drier,
  filter, sight glass, accumulator)

Save sim outputs to `sim/refrigeration/` per the project folder layout.

## CFD analysis (OpenFOAM via WSL)

Acceptance criteria (IEC matrix):
- Damp Heat: 85 °C ± 2 °C, 85 % RH ± 3 %, uniformity ±3 °C across module plane.
- Thermal Cycle: ramp ≤ 100 °C/h; no overshoot > 2 °C.
- UV mode: 60 °C ± 5 °C at module backside.

Cases to run sequentially:
1. DH steady state (UV off).
2. TC mid-ramp transient (heating).
3. TC mid-ramp transient (cooling).
4. UV-on steady state.

Folder structure (locked): `08-engineering/PV_TestLab/05_4in1_UV_TC_HF_DH/sim/cfd/case1_DH/`.
